Todo List App(Hex_Softwares_To-do list app)
A simple, modern Todo List application built with HTML, CSS, and JavaScript. This app allows you to manage your daily tasks with full CRUD operations and automatic browser storage.

Features
✅ Add new tasks - Quickly add tasks with Enter key or button click
✅ Mark as complete - Toggle task completion status
✅ Edit tasks - Update existing task text
✅ Delete tasks - Remove tasks you no longer need
✅ Filter tasks - View All, Pending, or Completed tasks
✅ Automatic saving - Tasks are saved to browser's localStorage
✅ Statistics - See total, completed, and pending task counts
✅ Responsive design - Works on desktop and mobile devices
✅ Recent first ordering - Newest/modified tasks appear at the top

Installation

No installation required! Simply:
Download the project folder
Open the file index.html in your web browser
Start managing your tasks!
